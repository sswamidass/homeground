# How to use the page templates

This theme includes page templates matching the React routes. To use them:

1. In the WP admin, create a new Page (Pages -> Add New).
2. Choose the template from the Page Attributes -> Template dropdown (e.g., "About Page").
3. Add content in the editor — the template will render the content area.

Templates included:
- About Page (`page-about.php`)
- Books Page (`page-books.php`) — lists `book` CPT entries
- Monthly Box (`page-monthly-box.php`)
- Coffee (`page-coffee.php`)
- Subscription (`page-subscription.php`)
- FAQ (`page-faq.php`)
- Contact (`page-contact.php`)
